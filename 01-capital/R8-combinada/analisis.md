# R8 — Combinada escalonada 🏆

> **Dimensión:** Capital
> **Estado:** ✅ validada (es la Ruta Maestra)
> **Score ponderado:** 4.65 / 5

---

## Definición

La combinación secuenciada de todas las fuentes: R1 + R2 + R3 para arrancar ($32,000-45,000), R7 en paralelo para habilitar, R4/R6 para Fase 2.

## Números clave

| Métrica | Valor | Nota |
|---|---|---|
| Capital arranque | $32,000-45,000 | R1+R2(+R3) |
| Tiempo a primer ingreso | 3 semanas | |
| Deuda real asumida | ~$17,000-20,000 a 0% | Familia + PROMOVER |
| Riesgo máximo | $32,000 | Acotado |
| Probabilidad | ALTA | No depende de una sola fuente |

## Secuencia

```
SEMANA 1-2:  R1 (abuela+pagaré) + R2 (ventas) + R3 (registro PROMOVER) + R7 (reporte buró + quitas)
SEMANA 3:    Capital $32,000-35,000 → compra N1 → a facturar
MES 2-4:     Anticipo 50% → WC con flujo → quitas completadas
MES 6-8:     Buró limpio + métricas → R4 (socio) o R6 (NAFIN) para Fase 2 ~$230,000
```

## Pros y contras

| Pro | Contra |
|---|---|
| No depende de UNA fuente (si R3 cae, R1+R2 bastan) | Exige ejecutar 4 frentes en paralelo |
| Ninguna fuente consulta tu buró | Disciplina férrea de anticipos |
| Arranque en 3 semanas, riesgo acotado | |
